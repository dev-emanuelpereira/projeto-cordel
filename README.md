# Projeto Cordel - Efeito Parallax

Projeto desenvolvido com **HTML5** e **CSS3** com o objetivo de praticar o efeito visual **Parallax**, utilizando como base o cordel *"Cordel Moderno"*, de **Milton Duarte**.

**Acesse o site:**  
https://dev-emanuelpereira.github.io/projeto-cordel/

---

## Visão Geral

Este projeto apresenta um cordel exibido em formato de página web interativa, explorando o efeito **Parallax**, onde imagens de fundo se movem em velocidades diferentes em relação ao conteúdo, criando sensação de profundidade.

O desenvolvimento foi feito utilizando apenas **HTML e CSS puro**, sem uso de frameworks ou bibliotecas externas.

---

## Tecnologias Utilizadas

- **HTML5**
  - Estrutura semântica
  - Organização de conteúdo textual

- **CSS3**
  - Efeito Parallax
  - Background fixo
  - Responsividade
  - Tipografia personalizada

---

## Objetivos do Projeto

- Praticar criação do efeito **Parallax**
- Trabalhar com posicionamento e estilização avançada em CSS
- Desenvolver layout visualmente atrativo
- Explorar apresentação textual em formato web

---

## Responsividade

O projeto foi desenvolvido buscando boa adaptação para diferentes tamanhos de tela, proporcionando uma experiência consistente em desktop e dispositivos móveis.
